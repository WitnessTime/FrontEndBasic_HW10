• git branch - посмотреть список веток
  • git status - посмотреть текузие измения
  • git add .... - проиндексировать гитом
  • git commit -m <название комита> - создаем комит
  • git push origin <название ветки> - заливаем комит на гит сервер
  • git pull origin <название ветки> - забрать актуальные измения с ветки
